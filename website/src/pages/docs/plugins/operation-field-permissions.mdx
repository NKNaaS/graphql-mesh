# Operation Field permissions

You can use `@graphql-mesh/plugin-operation-field-permissions` plugin to restrict the scope of certain public API users to a subset of the public GraphQL schema.

Based on Envelop's [useOperationFieldPermissions](https://www.envelop.dev/plugins/use-operation-field-permissions)

This plugin and authorization on a resolver level (or via middleware) are complementary. See [authorization guide](/docs/guides/auth0) for more details.

### Getting Started

<PackageCmd packages={['@graphql-mesh/plugin-operation-field-permissions']} />

## Example Configuration

```yaml
# ...
plugins:
  - operationFieldPermissions:
      permissions:
        - if: 'context._auth0?.sub != null'
          allow:
            - '*'
```
